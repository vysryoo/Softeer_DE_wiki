

# [2026-07-13] 회고

## Review (진행한 일)
 - Apache Hadoop의 전체적인 아키텍쳐와 동작 흐름에 대해서 배웠다.
 - Hadoop에 뼈대가 되는 HDFS, YARN, MapReduce 시스템에 대해서 배우고 각 레이어가 어떤 역할을 하는지 배웠다
 - HDFS에 한 번 쓰여진 파일은 수정이 불가능한 WORM 특성에 대해서 배웠고, 이런 특징이 락에 대한 처리를 할 필요가 없게 만들어서 분산 처리에서 높은 처리량을 뽑아낼 수 있도록 만드는 것임을 알게 되었다.
 - YARN에서의 RM와 NM으로 이루어진 마스터-슬레이브 구성을 배웠고, 각 노드들이 어떤 흐름으로 통신하여 분산 처리를 만들어내는지 알게 되었다.
 - 무거운 데이터를 가져오는 대신에 가벼운 연산을 각 데이터 노드로 보내서 연산을 처리하는 MapReduce 방식에 대해서 배웠다.
 - 엄청나게 많은 로그 데이터(수정 불필요)를 저렴한 시스템에 나누어 저장하기 위해서 만들어진 HDFS의 등장 배경에 대해서 이해하고 어떤 시스템에서 사용해야할 지 생각해보았다.
 

## 잘한 점
 - WORM 특성이 분산 환경에서 락 오버헤드를 없애 줌으로써 시스템을 단순화한다는 개념을 이해하였다.


## 아쉬운 점
 - 처음 접하는 내용이라 각 구성 요소의 내용을 파악하는데 급급하여, 왜 이렇게 설계되었는지 까지는 생각하지 못하였다.
